# Browser MCP: базовый регрессионный prompt

Открой опубликованный сайт МИИИИПС и пройди следующие сценарии:
1. Главная: проверить верхнюю навигацию, бегущую строку и быстрые переходы.
2. Лекции и выступления: проверить, что показаны реальные публичные источники Татьяны Мунн и Timepad-маршрут.
3. Курс ЭИ: пройти каталог, программу, библиотеку и лекции.
4. Новости: проверить живые ленты, открытые публикации и переходы по источникам.
5. Вступление: проверить форму, каналы связи и статус отправки.
6. Кабинеты: автор, студент, научный руководитель.
7. Mobile pass: главная, лекции, курс ЭИ, новости, join.

Для каждой страницы зафиксируй:
- есть ли битые ссылки;
- есть ли дубль блоков;
- уходит ли пользователь в понятный следующий шаг;
- не ломается ли layout на mobile.
