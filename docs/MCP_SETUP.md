# MCP и внешние интеграции для МИИИИПС

## Что уже есть в текущей среде
Сейчас в Codex уже подключён `figma` MCP.

## Что нужно подключить следующим этапом
1. Browser MCP
   - автоклики;
   - формы;
   - smoke/regression testing;
   - скриншоты сценариев.
2. GitHub MCP
   - работа с репозиторием сайта после публикации в GitHub;
   - чтение документации и структуры проекта в реальном времени.
3. Google-интеграции
   - Sheets;
   - Docs;
   - Gmail;
   - YouTube Data API.

## Важно
MCP не подключаются «внутрь сайта». Они настраиваются в среде Codex/клиента.
Поэтому в этой версии сайта мы:
- готовим архитектуру и документацию;
- делаем фронт и локальный API совместимыми с будущими MCP-проверками;
- выносим GitHub Pages в отдельный репозиторий для дальнейшего GitHub MCP.

## Что уже подготовлено на стороне сайта
- GitHub Pages workflow;
- локальный API-контур для форм;
- страницы и сценарии, которые Browser MCP сможет гонять автоматически;
- карта ролей и smoke-сценариев для кабинетов.
