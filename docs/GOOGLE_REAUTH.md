# Повторная авторизация Google для форм сайта

Когда формы должны реально писать в Google Sheets и отправлять email, нужно обновить текущие токены.

## Что сейчас происходит
- локальный API работает;
- формы пишутся в локальный JSONL;
- интеграции Gmail / Google Sheets готовы кодом;
- но старые токены оказались revoked / invalid.

## Что нужно сделать
1. Запустить `google_reauth_helper.py`.
2. Подтвердить вход в браузере под нужным Google-аккаунтом.
3. Разрешить доступ к Gmail и Google Sheets.
4. После этого формы начнут работать не только локально, но и писать в Google-контур.
