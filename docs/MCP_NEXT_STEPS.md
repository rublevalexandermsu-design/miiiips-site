1. Browser MCP
- автоклики по страницам

- проверка форм и мобильных сценариев

2. GitHub MCP
- чтение/контроль репозитория в реальном времени
- работа с issue/PR/repo metadata
- управление дальнейшими публикациями

3. Google / YouTube MCP
- чтение табличных реестров
- работа с контентом лекций и лентой обновлений

## Что это даст
- меньше ручного прокликивания
- быстрее ловим поломки
- проще поддерживать сайт после каждой волны изменений
