# Agent Task Templates

## Назначение

Готовые короткие постановки задач для каждого агента.

---

## Global Orchestrator

### Шаблон
Разбери задачу по доменам, определи какие агенты нужны, какие страницы и data files затрагиваются, где есть риск конфликта, и верни порядок исполнения и release path.

---

## Architecture Agent

### Шаблон
Проверь задачу на уровне платформы: какие shared entities, shared routes, shell-изменения и cross-project связи она затрагивает. Скажи, что должно жить в shell, а что — в модуле.

---

## Accounts Agent

### Шаблон
Обнови participant shell и ролевые страницы так, чтобы задача корректно отражалась в статусах участника, маршрутах и правах доступа.

---

## Events Agent

### Шаблон
Обработай задачу как часть жизненного цикла мероприятия: страница события, календарь, карточка, материалы, обновление архива, без утечки служебного текста в публичный слой.

---

## Courses Agent

### Шаблон
Обнови курс как часть фабрики курсов: страница, модули, прогресс, доступ, сертификат, оплата и следующий шаг участника.

---

## Grants Agent

### Шаблон
Обнови грантовый маршрут: релевантность по профилю, стадия готовности, шаг заявки, доказательная база и роль института.

---

## Editorial Agent

### Шаблон
Преврати входящий материал в human-facing публикационный результат: чистый публичный текст, структура, связи с курсом/событием/кейcом и готовность к выпуску.

---

## Research Agent

### Шаблон
Оформи исследовательский материал как research case: артефакты, стадия, проверка, публичная страница, связи с курсом, грантом и публикацией.

---

## Compliance Agent

### Шаблон
Обнови внутренний контур отчётности: перечень обязательных сроков, месячную навигацию, Telegram mirror, архив неактуальных периодов, папки доказательств и статусы подач, не ломая правовой контур и источник данных.

---

## QA Agent

### Шаблон
Проверь публичный выпуск: техтекст, заглушки, route/data consistency, пустые блоки, разрывы шаблона и выпускную готовность.
